Week 1-2: Programming Basics

  • Topics:
    • Variables, loops, condititonal statements, arrays, functions, recursion.
    • Introduction to time complexity (Big-O, Big-Theta, Big-Omega).
  • Resources:
    • Free Code Camp- JAVA for beginners
  • Practice:
    • Solve Basic problems on HackerRankor CodeChef

Week 3-4: Recursion and Time-complexity basics

  • Topics:
    • Recursive algorithms, base case, and recurrence relations.
    • Time complexity of loops and recursive functions
  • Resources:
    • Abdul Bari- recursion basics
    • GFG- Time Complexity Analysis
  • Practice:
    • Recursion problems on LeetCode

Week 5-6: Arrays & Strings

  • Topics:
    • Operations on arrays (traversal, insertion, deletion)
    • Prefix sums, sliding window, two-pointer techniques
    • Basic string manipulations
  • Resources:
    • Take U Forward - Arrays Playlist
    • Love Babbar- String Problems
  • Practice:
    • Solve Problems on GFG Arrays Problems
    • Easy String Problems on LeetCode

Week 7-8: Linked-Lists

  • Topics:
    • Singly, doubly, and circular linked lists.
    • Operations: insertion, deletion, reversal.
  • Resources:
    • Apna College- Linked-Lists tutorial
    • Take U Forward- Linked-Lists Playlist
  • Practice:
    • Linked-Lists problems on geeksforgeeks

Week 9-10: Stacks & Queues

  • Topics:
    • Stack operations and applications (e.g., parentheses matching, postfix evaluation)
    • Queue and deque operations.
  • Resources:
    • Take U Forward- Stacka & Queues Playlist
    • CodeWithHarry- Stacks and Queues
  • Practice:
    • Problems on GFG and LeetCode for Stacks & Queues

Week 11-12: Hashing

  • Topics:
    • Hash maps, hash sets, collision handling
    • Applications like frequency count, two-sum problem
  • Resources:
    • Abdul Bari- Hashing techniques
    • Take U Forward- Hashing
  • Practice:
    • Problems ont LeetCode & GFG

Week 13-14: Trees

  • Topics:
    • Binary trees, binary search trees (BST).
    • Tree traversals (inorder, preorder, postorder)
  • Resources:
    • Take U Forward- Trees Playlist
    • Apna College- Trees Basics
  • Practice:
    • Trees problems on GFG

Week 15-16: Heaps & Priority Queues

  • Topics:
    • Max-heap, min-heap, priority queue operations
    • Applications: Kth largest element, heap sort.
  • Resources:
    • Kunal Kushwaha's Heap DS and Priority Queues tutorial
    • Abdul Bari- Heap Data Structure
  • Practice:
    • Problems on LeetCode and GFG

Week 17-18: Sorting & Searching

  • Topics:
    • Sorting algorithms: Bubble, selection, merge sort, quicksort.
    • Searching algorithms: Binary search and its applications
  • Resources:
    • Love Babbar's Searching and Sorting Playlist
  • Practice:
    • Practice Searching & Sorting problems on LeetCode

Week 19-20: Dynamic Programming (DP)

  • Topics:
    • Fibonacci series, knapsack problem, longest common subsequence.
  • Resources:
    • Take U Forward- DP Series
    • Aditya Verma- DP Playlist
  • Practice:
    • Easy and Medium DP Problems on LeetCode

Month 6-8:

  • Topics:

  • Practice Startegy:

    • Weekend: Practice 1-2 medium/hard problems.
